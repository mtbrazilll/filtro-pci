Política de Privacidade

Última atualização: 21/04/2026

1. Visão geral

A extensão Filtro Avançado para PCI Concursos foi desenvolvida por Mateus da Silva para facilitar a filtragem visual de oportunidades em páginas compatíveis do site PCI Concursos.

2. Dados coletados

Esta extensão não coleta, não vende, não compartilha e não transmite dados pessoais para servidores externos.

3. Dados armazenados localmente

A extensão utiliza somente a API chrome.storage.local para salvar, no próprio navegador do usuário:

  • texto de inclusão
  • palavras de exclusão
  • escolaridade selecionada
  • salário mínimo
  • data mínima de inscrição

Esses dados permanecem apenas no navegador do usuário e não são enviados ao desenvolvedor nem a terceiros.

4. Permissões

  • storage: para salvar as preferências de filtro localmente.

Além disso, a extensão executa seu script apenas em páginas do domínio pciconcursos.com.br para inserir a interface de filtros e exportação CSV.

5. Compartilhamento com terceiros

Nenhum dado é compartilhado com terceiros.

6. Segurança

Como não há transmissão de dados para servidores externos, o processamento ocorre localmente no navegador do usuário.

7. Alterações nesta política

Esta política poderá ser atualizada para refletir melhorias, ajustes legais ou mudanças técnicas na extensão.
